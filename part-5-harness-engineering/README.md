# Part 5 — Harness Engineering

> **Phase 1: Get Dangerous** | 4 chapters | Beg-Adv | Language: TypeScript + Configuration

The meta-skill: making yourself and your team 10x more productive with AI tools. After Parts 1-4 you can build AI features. After Part 5 you make *everyone around you* more productive with AI.

This Part is different from what came before. Parts 1-4 taught you to build AI into your product. Part 5 teaches you to build AI into your *workflow*. The code examples here are configuration files, markdown templates, and automation scripts rather than application code. The principles are the same — tools, loops, memory, context — but the target is your own engineering process.

The four chapters follow a tight spiral:

1. **Master the tool** — Claude Code as an agent loop you control (Ch 23)
2. **Extend it** — MCP servers connect Claude Code to every system you use (Ch 24)
3. **Share and automate** — skills, plugins, and scheduled agents multiply your reach (Ch 25)
4. **See the industry** — coding agents, multi-agent delegation, and where this is all going (Ch 26)
